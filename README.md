# TubeReel Downloader

Phần mềm miễn phí (Windows) tải video YouTube và Facebook Reel, kèm tải script/phụ đề YouTube
(SRT có timestamp + TXT văn bản thuần).

## Tải xuống

👉 [**Tải TubeReelDownloader_Setup.exe**](../../releases/latest/download/TubeReelDownloader_Setup.exe)

Chạy file vừa tải để cài đặt — không cần cài Python hay ffmpeg riêng (đã được tích hợp sẵn).

## Tính năng

- Tải video YouTube và Facebook Reel, chọn chất lượng (Tốt nhất / 1080p / 720p / 480p / 360p)
- Tải riêng âm thanh dạng MP3
- Tải script/phụ đề YouTube (SRT, TXT), chọn ngôn ngữ
- Miễn phí, không yêu cầu license
